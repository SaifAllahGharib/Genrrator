# Genrrator

تطبيق لانشاء ارقام عشوائيه عشريه.

## الميزات الرئيسية

- مكانيه تسجيل الدخول اذا كان لديك حساب.
- إمكانية إنشاء حساب اذا لم يكن لديك حساب.
- إمكانية الدخول لأنشاء ارقام ب كود معين.
